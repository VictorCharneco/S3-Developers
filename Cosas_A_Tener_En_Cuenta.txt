Para Iniciar el servidor en local utilizar este comando -> php -S localhost:8000 web/index.php

PASOS PARA CREAR NUEVA VISTA CON SU CONTROLADOR:

1er paso -> crear archivo "prueba.phtml" dentro de la carpeta app/views/scripts/"nombre_carpeta"/
	el "nombre_carpeta" aconsejo que se llame igual que el archivo por ejemplo app/views/scripts/prueba/prueba.phtml

2ndo paso -> Crear controlador en la carpeta app/controllers
	Aconsejo que el nombre del controlador empieze por el nombre Prueba en el caso de ejemplo y acabe en Controller , és decir , PruebaController.php

3r paso -> Declarar la ruta en app/config/routes.php 
	las rutas queda algo como asi : $routes = array(
					'/' => 'prueba#prueba',
					'/test' => 'test#index'
					);
	para que lo entendáis de la manera mas fácil  , primero declaramos la ruta http, en el caso de ejemplo será "/" esta ruta quedaría algo	asi 	"http://localhost:8000/" una vez puesta la ruta hemos de hacer 	referencia a los scripts que contendrá esta ruta.

	"prueba#prueba" -> esto hace referencia al nombre de la carpeta y el nombre del archivo dentro de la carpeta app/views/scripts/"nombre_carpeta"/
	si no ponemos los nombres iguales dará error , porque no encontrara la carpeta o el archivo.
	Ejemplo -> app/views/scripts/prueba/prueba.phtml esta estructura esta bien echa, en cambio si pongo app/views/scripts/pruebaDemo/prueba.phtml 
	debería poner en la ruta esto -> pruebaDemo#prueba


Acceder a Base de datos JSON : 

	He creado una variable GLOBAL llamada JSON_DATA_PATH, que contiene el path del archivo json.
	
	json_decode -> pasa de json a array asociativa -> para conseguir informacion 
	json_encode -> te pasa la array a formato JSON -> para insertar cambios
	file_put_contents -> añade los cambios al archivo json
